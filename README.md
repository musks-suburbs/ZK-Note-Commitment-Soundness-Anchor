# ZK Note Commitment + Soundness Anchor

## Overview
Этот репозиторий демонстрирует минимальный прототип механизма soundness, вдохновлённого Aztec, Zama и другими ZK-проектами.  
Скрипт создаёт коммитмент (хэш) приватной «записки» и закрепляет его на доверенном слое, получая хэш блока L1 для проверки целостности.

## Установка
1. Установите Python 3.10+
2. Установите зависимости:
   pip install web3
3. В файле app.py замените your_api_key на свой ключ Infura или RPC провайдера.

## Запуск
   python3 app.py

## Ожидаемый результат
Программа:
- создаёт локальный коммитмент заметки;
- получает хэш блока Ethereum L1;
- выводит оба значения;
- сообщает, что привязка к L1 выполнена и soundness обеспечен.

## Примечания
- Механизм коммитмента аналогичен идее private notes в Aztec.
- Можно заменить структуру заметки на собственные поля.
- Блок для привязки можно выбрать любой; звукость обеспечивается благодаря детерминированности хэшей.
